# Jest Go to Test extension for VSCODE

Jump between code and test in projects using Jest.

## Default keybinding:

* Ctrl + Shift + y
* Cmd + Shift + y (Mac)

## Redine shortcuts:

In keybindings.json

```
  ...
	{
		"key": "shift-cmd-y",
		"command": "extension.jestGoToTest",
		"when": "editorFocus"
	}
	...
```
